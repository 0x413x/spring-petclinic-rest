Root URL for REST API: /petclinic/api

HTTP method URL Description
Owner entity
GET /owners Get list all owners
POST /owners Insert new owner
GET /owners/:id Get owner by :id
PUT /owners/:id Update owner with :id
DELETE /owners/:id Delete owner with :id
Pet entity
GET /pets Get list all pets
POST /pets Insert new pet
GET /pets/:id Get pet by :id
PUT /pets/:id Update pet with :id
DELETE /pets/:id Delete pet with :id
